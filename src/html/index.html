Welcome to Gamma Web Server! 

Congratulations on setting up Gamma Web Server! This page is being served to you by Gamma Web Server instance and is the default page that is displayed when you launch the server for the first time.

What's Next?

Now that you have Gamma Web Server up and running, you can serve static files such as HTML, CSS, and JavaScript from a specific directory, usually the "html" directory if the directory is not changed within the code.

Editing the Website Content

To edit the content of your website, you can modify the static files in the "html" directory or the handler functions in your Go code. The static files can include HTML, CSS, and JavaScript files, as well as images and other media. You can use a text editor or an integrated development environment (IDE) to edit the files.
